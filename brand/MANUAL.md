# Brand Manual v0.1

<!-- HUMAN NOTE: This is a working manual, not a precious identity book. The mark should be easy to print, steal, patch, remix, and recognize from distance. -->

Full City Columbus uses Earthwork Futurism:

old ground,
new machine,
public myth you can wear.

The system should feel civic, bright, severe, local, and alive.

## Order Of Impact

1. Mark
2. Line
3. Place
4. Demand
5. Scan path
6. Source trail

If the artifact starts by explaining itself, rebuild it.

## Core Mark

The Neo Columbus Signal:

- circle
- crossing
- ring
- amber core

Use it as:

- avatar
- sticker
- flag stamp
- QR badge
- poster mark
- field note cover
- public comment slide
- neighborhood module

Do not add skyline clutter, seal clutter, explorer imagery, mascot filler, or fake ancient costume.

## Flag

Primary flag:
Second Geometry.

Use a 3:2 ratio.

Elements:

- River Navy field
- Limestone axes
- Limestone ring
- Bronze center rule
- Transit Amber core

The flag should survive:

- phone avatar
- sticker
- cheap flyer
- patch
- banner
- bridge-distance recognition

## Type

Public commands:
heavy condensed display type.

Small labels:
technical sans, uppercase, tight but readable.

Body copy:
bold sans, short lines, enough air to read on mobile.

Do not use tiny light text on busy dark backgrounds.
Do not let all-caps body text become a wall.
Do not shrink words until the artifact looks scared.

## Layout

Use hard fields:

- flag block
- manual grid
- street poster
- field card
- proof wall
- mission label

Prefer:

- circle
- axis
- ring
- route line
- box
- field
- badge

Avoid:

- soft lifestyle gradients
- decorative blobs
- weak hero cards
- stock civic smiles
- fake official seals

## QR Rule

Campaign QR target:

https://neocolumbus.github.io/Project-Columbus/signal/

Use QR when the artifact needs a street-to-repo handoff.

Minimums:

- keep a clean quiet zone
- do not distort the code
- do not bury it under texture
- do not place it over important type
- test the scan before release

## Collision Test

Before shipping a visual asset:

- read it at phone size
- read it from across the room
- check all text stays inside its field
- check QR quiet zones
- check the mark is not buried
- check contrast on dark and light backgrounds
- check it still works in black and white

If the piece needs explanation before it hits, it is not done.

## Copy Law

Short beats clever.
Place beats abstraction.
Object beats essay.
Demand beats mood.

Good:

- Signal seen: Confluence.
- Report a dead wall.
- The machine pays tribute.
- Morse Road is not a fragment.
- Frequency is dignity.

Bad:

- transformative civic innovation
- inclusive placemaking framework
- authentic community activation
- comprehensive engagement ecosystem

The line should feel like it could be printed huge.
