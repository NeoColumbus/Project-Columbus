## The Neo Columbus Signal

The signal is the portable form of the Second Geometry:

- circle
- crossing
- ring
- amber core

It should work as a sticker, avatar, stencil, patch, poster stamp, folder icon, street mark, and GitHub repo badge.

## Meaning

The circle is gathering.
The crossing is the old center.
The ring is civic room.
The amber core is ignition.

The mark says:

The earthworks were first.
The next city is us.

## Usage Rules

Use the mark hard and often.

Do not make it precious.
Do not bury it under explanation.
Do not add a mascot, skyline, seal, explorer, or fake-official clutter.

Best uses:

- sticker
- QR sticker
- poster stamp
- profile icon
- field note cover
- social square
- neighborhood module
- public comment slide
- repo badge

The mark should feel like people found it before they understood it.

## QR Rule

Campaign QR codes should point to the scan path:

https://neocolumbus.github.io/Project-Columbus/signal/

Use them where the artifact needs a handoff from street to repo.
Keep the quiet zone clean.
Do not bury the code under texture, distortion, or tiny print.
