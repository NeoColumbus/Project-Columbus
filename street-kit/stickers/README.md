# Individual Sticker Pages

These files are letter-size printable pages with one poster sticker per page.
Each page inlines the poster art so GitHub can render the SVG preview.

Use the QR version when the object needs a scan path. Use the no-QR version
when the code will live beside it as a separate utility sticker.
