# Drop System

<!-- HUMAN NOTE: Think in releases, not one-off posts. Every drop should have a symbol, a line, a demand, and something people can use. -->

The project should move in drops.

Each drop needs:

1. a symbol
2. a line
3. a civic demand
4. a printable artifact
5. a source trail
6. an invitation to contribute

## Drop 01: Second Geometry

Symbol:
Neo Columbus Signal and Second Geometry flag.

Line:
The earthworks were first. The next city is us.

Action:
Post the signal. Name the break. Build the body.

Demand:
Treat housing, transit, water, power, and public beauty as civic form.

Artifacts:

- signal mark
- flag SVG
- evidence poster set
- sticker sheet
- post-the-signal handbill
- social squares
- vertical story frames
- bus stop audit card
- dead wall report card
- data center public test card
- neighborhood generator
- field note on ancient-to-modern Columbus

Source:
[../drops/drop-001-second-geometry.md](../drops/drop-001-second-geometry.md)

## Distribution

Every drop should be usable in:

- GitHub
- Instagram
- group chats
- printed flyers
- stickers
- public comment
- neighborhood meetings
- bus stops
- campus
- downtown windows

The same idea should move from file to street without losing force.
