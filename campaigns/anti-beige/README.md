# Anti-Beige

<!-- HUMAN NOTE: Use this campaign to attack ugliness, not residents. Add examples with care. Name bad patterns, not people with less money. -->

Ugliness is not neutral.

Blankness is not innocence.
Cheap dead frontage is not maturity.
Value-stripped design is not humility.

Beige development is a political and spiritual failure.

Anti-Beige does not mean anti-housing.
Build housing.
Build a lot of it.
Build it near transit.
Build it where people can live full lives.

But do not use the need for homes as an excuse to insult the city.

## Key Line

Build the city. Do not beige it to death.

## The Claim

Beauty is not extra.
The public realm is not extra.
Materials are not extra.
Windows are not extra.
Corners are not extra.
Ground floors are not extra.

Daily life deserves form.
