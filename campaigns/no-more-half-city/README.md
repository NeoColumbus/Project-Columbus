# No More Half-City

<!-- HUMAN NOTE: Replace abstraction with places. A stronger version should name routes, blocks, bridges, missing crossings, and rooms where Columbus becomes whole or breaks apart. -->

Columbus has mass but not always presence.

It has districts.
It has events.
It has campus weather.
It has suburban gravity.
It has downtown fragments.

But a pile of fragments is not a city.

No More Half-City is the campaign against civic partialness.

The question is not whether Columbus is growing.
The question is whether the whole body can be felt.

## Key Line

A city is when the whole body starts to move.

## Targets

- District-only identity.
- Event-only civic feeling.
- OSU-only emotional scale.
- Downtown treated as decoration.
- Suburbs treated as exile.
- Transit treated as charity.
- Housing treated as an inconvenience.

## Demand

Make Columbus legible as one body.

Not one taste.
Not one neighborhood.
Not one class.
Not one skyline angle.

One body.
Many rooms.
One pulse.

## What This Builds

This campaign turns scattered civic feeling into one public image.

It should show up anywhere Columbus gets described too small:

- neighborhood meetings
- downtown planning
- transit fights
- housing fights
- county-city coordination
- festival copy
- sports-adjacent civic energy
- regional growth debates

The point is not to erase difference.
The point is to stop letting difference become civic amnesia.

## Public Demands

- Treat downtown, neighborhoods, suburbs, campus, and corridors as one civic body.
- Make regional transit feel like a unifying project, not a service category.
- Build public spaces that connect districts instead of branding them into islands.
- Stop letting OSU carry all large-scale emotional identity.
- Make the capital city visible outside government buildings.
- Tie housing, transit, safety, downtown, and design into one public story.

## First Builds

- Make a neighborhood-specific Half-City flyer.
- Make a corridor map showing where the body breaks.
- Add a photo set of places where Columbus feels whole.
- Add a photo set of places where Columbus gets severed.
- Write a public comment template using the line: "A pile of projects is not a city."
