# No More Half-City

<!-- HUMAN NOTE: Replace abstraction with places. A stronger version should name routes, blocks, bridges, missing crossings, and rooms where Columbus becomes whole or breaks apart. -->

Columbus has mass but not always presence.

It has districts.
It has events.
It has campus weather.
It has suburban gravity.
It has downtown fragments.

But a pile of fragments is not a city.

No More Half-City is the campaign against civic partialness.

The question is not whether Columbus is growing.
The question is whether the whole body can be felt.

## Key Line

A city is when the whole body starts to move.

## Targets

- District-only identity.
- Event-only civic feeling.
- OSU-only emotional scale.
- Downtown treated as decoration.
- Suburbs treated as exile.
- Transit treated as charity.
- Housing treated as an inconvenience.

## Demand

Make Columbus legible as one body.

Not one taste.
Not one neighborhood.
Not one class.
Not one skyline angle.

One body.
Many rooms.
One pulse.
