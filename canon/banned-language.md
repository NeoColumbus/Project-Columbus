# Banned Language

These phrases are not forbidden because they are evil.
They are forbidden because they make the city disappear.

Do not use them in public copy unless you are naming what must be destroyed.

- stakeholder engagement
- placemaking
- vibrant ecosystem
- community-forward
- innovation hub
- cross-functional
- alignment
- leverage
- activate
- strategic framework
- shared visioning
- robust process
- public-private partnership
- dynamic urban experience

## Replacement Rule

Name the thing.
Name the street.
Name the bus.
Name the building.
Name the signal.
Name the beauty.

Do not hide behind phrases that can survive anywhere.
