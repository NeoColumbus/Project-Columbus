# Colors

<!-- HUMAN NOTE: These colors are first defaults. Designers should test accessibility, print behavior, fabric, paint, and cheap flyer reproduction. -->

The palette avoids OSU scarlet and gray as the primary identity.

It belongs to river, stone, bronze, clay, transit, and night.

## Core Palette

| Name | Hex | Use |
| --- | --- | --- |
| River Navy | `#071923` | Primary dark ground, hero overlay, civic night |
| Deep Current | `#0E2A32` | Secondary dark, panels, footer |
| Limestone | `#E8DDC7` | Main text on dark, paper ground |
| Bone | `#F6F0E4` | Light page ground |
| Bronze | `#B77836` | Rule lines, emphasis, seals |
| Oxidized Copper | `#4F8A74` | Secondary accent, maps, quiet marks |
| Brick Clay | `#9D4732` | Campaign heat, print accents |
| Transit Amber | `#F4B037` | Calls to action, transit lines, signal color |
| Civic Black | `#050606` | Type, deep contrast |
| Dark Green | `#17352F` | Optional civic depth |

## Use

River Navy carries the night.
Limestone carries the word.
Transit Amber carries motion.
Bronze carries ceremony.
Brick Clay carries the street.
Oxidized Copper carries time.

Do not let the system collapse into one color.
Columbus is not a beige monochrome.

## Flag Defaults

The primary Second Geometry flag uses:

- River Navy field
- Limestone axes and ring
- Bronze center rule
- Transit Amber core
