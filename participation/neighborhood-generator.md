# Neighborhood Generator

<!-- HUMAN NOTE: The neighborhood system should let people localize the myth without turning it into bland civic copy. -->

## Base Spell

`[PLACE] is not a fragment.`

## Build The Line

1. Name the place.
2. Name what it already has.
3. Name what is missing.
4. Attach one demand.
5. End with the city body.

## Template

`[PLACE] is not a fragment. It has [ASSET]. It needs [MISSING PIECE]. Build the body.`

## Examples

`Linden is not a fragment. It has memory, streets, and pressure. It needs safer crossings and more public rooms. Build the body.`

`Franklinton is not a fragment. It has river, art, and heat. It needs homes people can stay in. Build the body.`

`Morse Road is not a fragment. It has the whole world moving through it. It needs shelter, shade, and frequency. Build the body.`

`Downtown is not a fragment. It has towers and state power. It needs residents, rooms, and night life. Build the body.`

## Keep It Honest

Use real streets.
Use real routes.
Use real landmarks.
Use real pressure.

Do not flatten neighborhoods into vibes.
Do not invent local authority you do not have.
Do not make poverty into decoration.
Do not turn pride into a brochure.
