# Post The Signal

<!-- HUMAN NOTE: This is the basic public ritual. Keep it easy enough that a novice can do it from a phone. -->

## The Loop

Post the signal.
Name the break.
Build the body.

## What To Post

Use one asset from Drop 001:

- [signal mark](../brand/signal-mark.svg)
- [poster set](../street-kit/posters)
- [sticker sheet](../street-kit/sticker-sheet.svg)
- [social squares](../social-kit/squares)

Then attach it to one Columbus truth.

## Caption Templates

`[PLACE] is not a fragment. The earthworks were first. The next city is us.`

`Signal seen: [PLACE]. Name the break. Build the body.`

`[PLACE] needs [MISSING PIECE]. A pile of projects is not a city.`

`The machine lives here. The machine pays tribute. Water, power, land, tax, compute, data.`

`Report a dead wall: [PLACE]. Density deserves beauty.`

## Good Places To Start

- a bus stop with no shelter
- a dead wall on a busy street
- a parking lot where city life should be
- a beautiful corner people ignore
- a public room that already works
- a route that should connect better
- a development that needs more civic beauty
- a data center or utility project that owes public benefit

## How To Send It Back

Open an issue or pull request with:

- place
- photo or short description
- the missing piece
- one proposed line
- one source if you have it

No expertise required.
Just make it real.
